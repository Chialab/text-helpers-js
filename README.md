## Install

```sh
$ npm install @chialab/text-helpers
```

## Development
### Requirements

* Node (>= 6)
* RNA cli (`npm install @chialab/rna-cli`)


### Build the project.

```
$ rna install
$ rna build --production
```

### Watch the project.
```
$ rna install
$ rna build --watch
```
